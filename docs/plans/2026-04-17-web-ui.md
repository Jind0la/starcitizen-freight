# Web UI Implementation Plan — Freight

> **For Hermes:** Use subagent-driven-development skill to implement this plan task-by-task.

**Goal:** Add a modern web UI to the existing Freight CLI app. The binary serves a web interface at `http://localhost:8080` when run with `--web` flag.

**Architecture:** Axum HTTP server serves:
1. Static files (HTML/CSS/JS) embedded in the binary
2. `/api/routes?scu=<cargo_scu>` — JSON endpoint that mirrors the CLI calculation logic

**Tech Stack:** Axum + tower-http (static files), vanilla HTML/CSS/JS (no framework), same Rust backend logic

## Task 3: Add Web Server Module

**Objective:** Create Axum HTTP server with API endpoint and static file serving

**Files:**
- Create: `src/web_server.rs`

**Step 1: Create `src/web_server.rs`**

```rust
use axum::{
    routing::{get, Router},
    serve,
    extract::{Query, State},
    response::{Html, Json, IntoResponse},
    http::StatusCode,
};
use std::net::SocketAddr;
use std::sync::Arc;
use tower_http::cors::CorsLayer;
use rust_embed::Embed;

use crate::api::UexClient;
use crate::calculation::rank_routes;
use crate::models::RankedRoute;
use crate::error::AppError;

// Embed static files
#[derive(Embed)]
#[folder = "src/web"]
struct StaticAssets;

// ---------------------------------------------------------------------------
// API Types
// ---------------------------------------------------------------------------

#[derive(serde::Deserialize)]
pub struct RoutesQuery {
    scu: u32,
}

#[derive(serde::Serialize)]
pub struct RoutesResponse {
    routes: Vec<RankedRoute>,
    total_fuel_estimate: f64,
    last_updated: String,
}

#[derive(serde::Serialize)]
pub struct ErrorResponse {
    error: String,
}

// ---------------------------------------------------------------------------
// Static file handler
// ---------------------------------------------------------------------------

fn static_handler(filename: &str) -> impl IntoResponse {
    match StaticAssets::get(filename) {
        Some(file) => {
            let mime = mime_guess::from_path(filename)
                .first_or_octet_stream();
            ([(axum::http::header::CONTENT_TYPE, mime.as_ref())], file.data.into_owned())
        }
        None => {
            // Serve index.html for SPA routing
            if let Some(index) = StaticAssets::get("index.html") {
                ([(axum::http::header::CONTENT_TYPE, "text/html")], index.data.into_owned())
            } else {
                (StatusCode::NOT_FOUND, "Not found".as_bytes().to_vec())
            }
        }
    }
}

// ---------------------------------------------------------------------------
// Routes
// ---------------------------------------------------------------------------

async fn routes_handler(
    State(client): State<Arc<UexClient>>,
    Query(query): Query<RoutesQuery>,
) -> Result<Json<RoutesResponse>, AppError> {
    if query.scu == 0 || query.scu > 16000 {
        return Err(AppError::InvalidInput("SCU must be between 1 and 16000".into()));
    }

    let (routes_result, fuel_result) = tokio::join!(
        client.get_routes(),
        client.get_fuel_prices()
    );

    let routes = routes_result?;
    let fuel_prices = fuel_result.unwrap_or_default();

    let ranked = rank_routes(&routes, &fuel_prices, query.scu);
    let total_fuel: f64 = ranked.iter().map(|r| r.fuel_cost).sum();

    Ok(Json(RoutesResponse {
        routes: ranked,
        total_fuel_estimate: total_fuel,
        last_updated: chrono::Utc::now().format("%H:%M UTC").to_string(),
    }))
}

async fn static_index() -> impl IntoResponse {
    static_handler("index.html")
}

async fn static_files(Path(filename): Path<String>) -> impl IntoResponse {
    static_handler(&filename)
}

// ---------------------------------------------------------------------------
// Server startup
// ---------------------------------------------------------------------------

pub async fn start_server(port: u16, token: Option<String>) -> anyhow::Result<()> {
    let client = Arc::new(UexClient::new(token));

    let app = Router::new()
        .route("/api/routes", get(routes_handler))
        .route("/web/app.js", get(|p| static_handler("app.js")))
        .route("/web/styles.css", get(|p| static_handler("styles.css")))
        .route("/", get(static_index))
        .with_state(client);

    let addr = SocketAddr::from(([0, 0, 0, 0], port));
    println!("🌐 Freight Web UI: http://localhost:{port}");
    println!("   Press Ctrl+C to stop");

    let listener = tokio::net::TcpListener::bind(addr).await?;
    serve(listener, app).await?;

    Ok(())
}
```

Add `--web` flag detection using `std::env`:

```rust
mod web_server;

#[tokio::main]
async fn main() -> Result<()> {
    dotenvy::dotenv().ok();
    
    let args: Vec<String> = std::env::args().collect();
    
    if args.contains(&"--web".to_string()) {
        // Web server mode
        let port = args.iter()
            .position(|a| a == "--port")
            .and_then(|i| args.get(i + 1)?.parse().ok())
            .unwrap_or(8080);
        
        let token = std::env::var("UEX_API_TOKEN").ok();
        web_server::start_server(port, token).await?;
    } else {
        // TUI mode (existing logic)
        // ... existing code ...
    }
    
    Ok(())
}

```

**Step 2: Update `.gitignore`**

```

```

Wait, the `rust_embed` with `#[folder = "src/web"]` needs the files to exist at build time. The `StaticAssets` derive macro embeds the entire folder.

Since we want the files tracked in git but the embedding to work, we keep them in `src/web/` normally. The `rust_embed` crate embeds them at compile time.
